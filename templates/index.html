GIVEAWAYS
Create and manage giveaways
🎉 Create Giveaway

Use the bot command in your server:

!giveaway [minutes] [winners] [prize]
Example: !giveaway 60 1 Nitro Classic

Reroll: !reroll [message_id] [winners]

📖 How It Works
1. Use !giveaway in your server
2. Bot posts an embed with 🎉 reaction
3. Members react to enter
4. Bot picks random winner(s) when time's up
5. Use !reroll to pick new winners

ANNOUNCEMENTS
Broadcast messages to your server
📢 How to Announce

Use this command in your server — the bot sends a formatted embed to any channel:

!announce #channel Your message here
Example: !announce #announcements Server maintenance tonight at 10pm!
